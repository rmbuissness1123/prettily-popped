The signature section-heading lockup — eyebrow + Bodoni display title + optional script accent + lead.

```jsx
<SectionHeading
  eyebrow="What I offer"
  title="Balloons that make it"
  script="pop"
  lead="From a single centrepiece to a 16-foot garland — designed, delivered, and installed."
/>
<SectionHeading title="Let's talk" align="center" onDark />
```

`align`: `left`/`center`. `onDark` recolors text/accents for ink sections. `script` appends a Parisienne accent inline to the title.
