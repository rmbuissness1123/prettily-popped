The design system currently uses:

## Needed from the client
1. **Logo / wordmark** files (SVG + PNG) if an official one exists.
2. **Event photography** — the live gallery at prettilypopped.com/gallery has real photos of
   garlands, arches, columns and centrepieces. Drop them here and swap the placeholders in
   `ui_kits/website/` (Home hero, Gallery tiles, About portrait, ServiceCard `image` prop).
3. **Brand fonts**, if any are licensed (current fonts are Google Fonts substitutes).

Photos should read **warm, bright, colourful and joyful** to match the palette.
